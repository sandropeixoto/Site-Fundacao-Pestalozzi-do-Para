# Fundação Pestalozzi do Pará - Site Institucional

O Site Institucional da Fundação Pestalozzi do Pará é uma plataforma moderna e acessível, desenvolvida para apresentar a missão, os valores e os serviços pedagógicos da primeira instituição especializada em Educação Especial do Pará. Com um design "Clean Minimalist", a interface transmite profissionalismo, acolhimento e a rica história da fundação que atua desde 1955.

## ✨ Funcionalidades Principais

> **Apresentação Institucional**  
> Seções fluidas e animadas que contam a história da fundação e detalham o processo de acolhimento e avaliação.

> **Trilha Pedagógica**  
> Apresentação clara dos programas educacionais oferecidos:
> - **PEI (Programa de Estimulação Inicial)**
> - **PEC (Programa de Estimulação Continuada)**
> - **PML (Programa de Multi Letramento)**

> **Design Responsivo e Mobile-First**  
> Experiência otimizada para todos os tamanhos de tela (desktop, tablet e mobile), com navegação acessível e menu hambúrguer dinâmico.

> **Performance Otimizada**  
> Arquitetura moderna focada em performance de carregamento rápido e fluidez visual nas interações.

## 🛠️ Stack Técnica

| Tecnologia | Descrição |
| :--- | :--- |
| **React** | Biblioteca principal para a criação das interfaces de usuário utilizando abordagens modernas de componentes. |
| **TypeScript** | Adiciona tipagem estática, garantindo um código mais seguro e de fácil escalabilidade. |
| **Vite** | Ferramenta de build de alta velocidade e servidor de desenvolvimento em tempo real. |
| **Tailwind CSS** | Framework utility-first utilizado para criar designs responsivos e customizados sem poluir o ambiente com folhas de estilo extensas. |
| **Framer Motion** | Biblioteca unificada de animações que proporciona transições suaves e responsivas por toda a interface. |
| **Lucide React** | Conjunto de ícones consistentes, leves e com design polido. |

## 🚀 Como Executar o Projeto

Para visualizar ou customizar este projeto em seu ambiente local, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/SandroPeixoto/Site-Fundacao-Pestalozzi-do-Para.git
   ```

2. **Acesse o diretório do projeto:**
   ```bash
   cd Site-Fundacao-Pestalozzi-do-Para
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. O servidor iniciará e você poderá acessar a aplicação em seu navegador padrão através de `http://localhost:3000` (ou a porta indicada no terminal).

Para gerar o build de produção:
```bash
npm run build
```
